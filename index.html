LifeHub

🔗 Live 💻 Code

LifeHub is a full-stack health & wellness platform: a vanilla ES6 + Tailwind SPA that renders Canvas-based analytics and works offline-first with local persistence and eventual sync to a Node/Express API. The service exposes versioned REST contracts (/api/v1) with strict request/response schema validation, idempotent writes, and a normalized error envelope; JWT auth and CORS hardening enforce per-user isolation; structured JSON logging and a /health probe support ops. Consistent resource models span nutrition, water, exercise, and mood. CI/CD ships from GitHub to Vercel with environment-scoped config and SPA routing; the repo is Docker-ready for multi-stage builds and a clean lift to managed Postgres with pooled connections as usage scales.
